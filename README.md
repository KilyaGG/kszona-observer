# 📊 KS ZONA ONLINE OBSERVER

Инструмент для мгновенного получения подробной информации о любом аккаунте в игре **КС Зона Онлайн**. Без регистраций, ввода паролей и полностью бесплатно!

---

## 🚀 Быстрая установка (Всего 3 шага)

### **Шаг 1. Установите менеджер скриптов**
Установите официальное расширение **[Tampermonkey](https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)**.  
*Оно необходимо, чтобы код автоматически запускался вместе с игрой. Расширение абсолютно безопасное и проверенное (более 10 млн скачиваний).*

### **Шаг 2. Установите основной скрипт**
Нажмите на кнопку ниже и в открывшемся окне Tampermonkey нажмите **«Установить»** (Install):

## 🎮 Как этим пользоваться?

1. **Запустите игру** «КС Зона Онлайн» на платформе Яндекс Игры.
2. **Активируйте режим поиска**:
   * Нажмите на синюю кнопку **📊 Профиль** в верхнем меню хедера Яндекс Игр.
   * *Или* нажмите клавишу **[F4]** на клавиатуре.
3. **Ожидайте подключение**: откроется новая вкладка, которая перейдет в режим ожидания игровых данных.
4. **Посмотрите профиль**: вернитесь в игру и откройте профиль нужного вам игрока.
5. **Готово!** Все скрытые данные и подробная статистика аккаунта мгновенно отобразятся на открывшейся ранее вкладке.

---

## ⚠️ Решение возможных проблем

* **Вкладка открывается пустой или не открывается вообще?**  
  Перед нажатием клавиши **[F4]** обязательно кликните мышкой в любое место по экрану самой игры, чтобы браузер понял, что действие совершаете вы. Также убедитесь, что в браузере не включена жесткая блокировка всплывающих окон для сайта Яндекс Игр.
