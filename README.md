# 📊 Захват профиля для Яндекс Игр

Скрипт для автоматического перехвата сетевых данных профиля в Яндекс Играх и вывода их в удобный дашборд.

## 🚀 Инструкция по установке

Для корректной работы расширения необходимо установить **оба** скрипта ниже:

1. [👉 Установить Скрипт №1: Кнопка интерфейса](СЮДА_ВСТАВЬТЕ_RAW_ССЫЛКУ_НА_ПЕРВЫЙ_СКРИПТ)
2. [👉 Установить Скрипт №2: Перехватчик данных](СЮДА_ВСТАВЬТЕ_RAW_ССЫЛКУ_НА_ВТОРOЙ_СКРИПТ)

> **Важно:** После перехода по ссылкам нажмите кнопку **«Установить»** (Install) в открывшемся окне Tampermonkey.

## 🎮 Как использовать
1. Откройте любую игру на платформе Яндекс Игры.
2. Нажмите кнопку **📊 Профиль** в верхнем меню или клавишу **F4** на клавиатуре.
3. Откроется новая вкладка, ожидающая данные.
4. Сделайте любое сетевое действие в игре (например, зайдите в профиль или магазин внутри игры).
5. Данные автоматически отобразятся на открывшейся вкладке!
